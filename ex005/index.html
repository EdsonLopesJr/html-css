Formatação de texto HTML5

Semântico e Forma

O uso das tags em HTML5 tem um uso semântico, as tags tem um significados para os conteúdos, diferente do HTML4 algumas tags que funcionava como forma do html4 foram obsoletos, por motivo de ser usado como forma não semântico(com significados).

Negrito / Destaque

As tags que deixam a letras em negritos como <b>,</b> é usado como forma, por isso não semântico e não são recomendados para uso, as tags recomendadas semântico são as tags Destaque <strong>,</strong>.

Itálico / Ênfase

Outra tags que também deixaram de ser usado por motivo de não ser semântico foram as tags <i>, </i> para deixar textos itálicos, o jeito recomendado para uso é as tags Ênfase <em>, </em>.

Texto marcado

Para marcar uma frase ou palara é utilizado tag <mark>texto marcado</mark> e para podemos configurarmos utilizamos a ferramenta CSS para alterar a cor em outras palavras mudar a forma, que seria a sua principal caracteristica.

Codigo-fonte / PRE-Formatação

A tag codigo fonte é representado pela tag 'code' e pra deixar pre formatado é usado a tag 'pre' exemplo.:
